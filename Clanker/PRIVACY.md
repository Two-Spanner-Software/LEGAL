# Clanker – Privacy Policy

**Last Updated:** 30th March 2026

Hey 👋  
This Privacy Policy explains what data Clanker collects, how it is used, and how it is handled.

By using Clanker, you agree to this policy.

---

## 1. What Data Clanker Collects

Clanker only stores data necessary for the bot to function properly.

This may include:

- Server IDs  
- Channel IDs  
- User IDs  
- Command-related data (such as balances, settings, counting stats, etc.)

Clanker does **not** collect personal information like real names, addresses, passwords, or private messages.

---

## 2. How Data Is Used

Stored data is used strictly for bot functionality, including:

- Running commands  
- Saving server settings  
- Maintaining economy systems  
- Tracking bot-related features  

Your data is never used for advertising or marketing purposes.

---

## 3. Data Sharing

Clanker does **not** sell, rent, or trade user data.

Data is not shared with third parties except where required by law.

---

## 4. Data Storage & Security

Data is stored securely and only retained as long as needed for bot features.

While reasonable efforts are made to protect stored data, no system can be guaranteed 100 percent secure.

---

## 5. Data Removal

You may request data removal by:

- Removing Clanker from your server  
- Contacting support  

Some data may be automatically removed when the bot leaves a server.

---

## 6. Policy Updates

This Privacy Policy may be updated over time.

Continued use of Clanker means you accept any updates.
